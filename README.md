# [GraphQL with React Tutorial](https://www.youtube.com/watch?v=YyUWW04HwKY&t=709s)

## Tech Used
- React
- Express
- GraphQL
- Apollo

## Set Up Server

### Install Dependencies
- `npm i express`
- `npm i graphql`
- `npm i express-graphql`

## Set Up React
- `npx create-react-app client`

In **/client**
- `npm i @apollo/client`
- `npm i graphql`
